# 添加咖啡豆页面 - UI 设计文档

- **类型**: 表单录入页面

---

## 设计风格

## 输入框样式

- **背景**: #FFFFFF
- **边框**: 1px solid #E8E2DA
- **圆角**: 20px
- **内边距**: 20px 24px
- **标签**: 14px, 字重600, #3D2B1F
- **输入**: 16px, 字重400, #3D2B1F
- **占位符**: #A9A29A
- **下方间距**: 16px

### 聚焦状态
- **边框**: 2px solid #7B3F00
- **阴影**: 0 0 0 4px rgba(123, 63, 0, 0.08)
- **过渡**: 200ms ease-out

### 已填写状态
- **背景**: #FDF8F3 (轻微暖色)
- **标签颜色**: #7B3F00

## 顶部导航

- **返回**: ← 箭头 + 文字，点击缩放0.95，页面右滑退出
- **保存按钮**: 
  - 背景 #7B3F00，文字白色
  - 圆角 14px，内边距 10px 20px
  - 必填项未填时禁用 (透明度50%)

---

## 动画效果

### 页面入场
- 输入框依次淡入上滑
- 每个延迟50ms
- 时长400ms，ease-out

### 聚焦动画
- 边框颜色过渡 200ms
- 阴影出现过渡 200ms

### 保存成功
- 按钮变绿色 #7D9A78
- 显示 ✓ 图标
- 1秒后自动返回

---

## 响应式

- 移动端: 宽度100%，两侧内边距20px
- 桌面端: 最大宽度480px，居中

---

## 验收标准

1. 输入框依次动画入场
2. 聚焦/填写有明显视觉反馈
3. 下拉选择流畅
4. 图片上传支持预览
5. 保存按钮根据必填项启用/禁用
6. 保存成功有绿色成功提示
7. 风格与现有项目一致 (主色#7B3F00)
