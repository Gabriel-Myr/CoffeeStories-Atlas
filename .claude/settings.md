# Claude Code - 系统级配置

本文件定义与 AI 助手交互的系统级规则和偏好设置。

## 响应语言

- 使用中文与用户交流
- 代码注释使用中文
- 保持语言一致性

## 代码风格

- **不主动添加注释** - 只在用户明确要求时添加注释
- **不创建文档文件** - 只在用户明确要求时创建 README 或其他文档
- **使用现有模式** - 遵循项目现有的代码风格和约定

## 文件操作规则

- 修改文件前先读取文件内容
- 优先编辑现有文件，而非创建新文件
- 删除文件前确认文件存在

## 任务管理

- 对于复杂任务，使用 TodoWrite 工具创建任务列表
- 标记任务进度：pending / in_progress / completed
- 一次只处理一个 in_progress 任务

## 与用户交互

- 避免过于主动的行动，只在用户明确要求时执行
- 使用 AskUserQuestion 工具询问用户以澄清需求
- 完成任务后运行 lint 和 typecheck 命令（如果项目有配置）

## 代码引用

- 使用 Markdown 链接格式引用代码
- 链接文本使用文件/函数/类的 basename
- 链接格式：`[link text](file:///absolute/path/to/file#L123-L145)`
